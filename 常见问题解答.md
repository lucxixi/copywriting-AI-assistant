# ❓ 文案AI助手 - 常见问题解答（2025最新版）

## 🚀 快速解决问题

### 🔧 配置与API相关

**Q1: 如何配置AI API？**  
A:  
1. 点击左侧“系统设置”  
2. 在“API Key配置”卡片中选择服务商（支持Google Gemini、OpenRouter、Together、SiliconFlow等）  
3. 输入API密钥，点击“保存”  
4. 可设置激活Key，系统自动调用激活Key进行生成  
5. 支持多Key管理与持久化

**Q2: 支持哪些AI服务商？**  
A:  
- Google Gemini（推荐，性能优秀）  
- OpenRouter（多模型选择）  
- Together.ai（高性价比）  
- SiliconFlow（国内访问友好）

**Q3: API密钥从哪里获取？**  
A:  
- Google Gemini: https://aistudio.google.com/apikey  
- OpenRouter: https://openrouter.ai/keys  
- Together.ai: https://api.together.xyz/settings/api-keys  
- SiliconFlow: https://cloud.siliconflow.cn/account/ak

**Q4: API连接失败怎么办？**  
A:  
- 检查网络连接  
- 检查API密钥是否正确  
- 检查服务商余额  
- 尝试更换其他服务商  
- 查看“系统设置”中的错误提示

### 📝 功能与使用问题

**Q5: 生成的文案质量不理想怎么办？**  
A:

- 尝试不同风格和参数

- 多生成几个版本进行选择

**Q6: 如何生成真实的微信群聊对话？**  
A:

- 选择产品、设置对话轮数  
- 可自定义角色、风格、关键词替换  
- 预览区支持一键复制和示例参考

**Q7: 产品分析结果不准确怎么办？**  
A:  
- 补充产品信息和图片  
- 优化产品描述  
- 多次分析取最佳结果  
- 可手动修正分析内容

**Q8: 如何批量处理内容？**  
A:  
- 文案生成：支持批量生成多个版本  
- 对话分析：可同时上传多个文件  
- 模板应用：批量应用到多个场景  
- 历史导出：支持批量选择导出

### 💾 数据与历史管理

**Q9: 数据存储在哪里？**  
A:  
- 所有数据本地存储于浏览器  
- 不会上传到服务器，保护隐私  
- 建议定期导出备份

**Q10: 如何备份和恢复数据？**  
A:  
1. 进入“系统设置”  
2. 选择“数据导入/导出”  
3. 点击“导出数据”下载备份  
4. 需要时通过“导入数据”恢复

**Q11: 数据丢失了怎么办？**  
A:  
- 检查是否清除了浏览器数据  
- 尝试从备份文件恢复  
- 查看历史记录中心是否有保存  
- 联系技术支持

**Q12: 如何清理存储空间？**  
A:  
- 删除不需要的历史记录  
- 使用“系统设置”一键清空功能  
- 导出重要数据后重置

**Q15: 数据安全吗？**

- 支持数据加密（本地模拟）  
- 可设置访问密码保护

**Q16: 如何设置关键词过滤？**  
A:  
1. 进入“系统设置”  
2. 在“关键词替换设置”中添加敏感词及替换词  
3. 支持批量添加、编辑、删除

**Q17: 生成内容会被保存吗？**  
A:  
- 仅保存在本地历史记录中心  
- 不会上传到云端  
- 可随时导出、清空

## 🆘 紧急问题处理

### 🚨 数据丢失紧急处理
1. 不要刷新页面
2. 检查浏览器历史记录
3. 查看本地存储数据
4. 尝试从备份恢复
5. 联系技术支持

### 🚨 功能完全无法使用
1. 清理浏览器缓存
2. 禁用浏览器扩展
3. 尝试无痕模式
4. 更换浏览器测试
5. 检查网络设置

### 🚨 API费用异常
1. 立即检查API使用量
2. 暂停自动生成功能
3. 检查是否有异常调用
4. 联系服务商确认
5. 调整使用策略

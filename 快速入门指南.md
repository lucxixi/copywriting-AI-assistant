# ⚡ 文案AI助手 - 快速入门指南

## 🎯 5分钟快速上手

### 第一步：配置AI服务 (2分钟)
1. 点击左侧导航栏底部的 **"系统设置"**
2. 在"API配置"标签页中：
   - 选择AI服务商（推荐Google Gemini）
   - 输入API密钥
   - 点击"测试连接"确认配置成功

> 💡 **提示**：如果没有API密钥，可以先使用默认配置进行体验

### 第二步：生成第一个文案 (2分钟)
1. 点击 **"文案生成"** 模块
2. 选择文案类型（如"产品推广"）
3. 填写基本信息：
   - 产品名称：例如"美白面膜"
   - 目标用户：例如"25-35岁女性"
   - 产品特色：例如"天然成分，快速美白"
4. 点击 **"生成文案"**
5. 复制满意的结果

### 第三步：保存为模板 (1分钟)
1. 在生成结果下方点击 **"保存为模板"**
2. 输入模板名称
3. 添加标签便于后续查找
4. 点击"保存"

🎉 **恭喜！** 您已经完成了基本操作流程

## 💡 使用技巧

### ✅ 提高生成质量的方法
1. **详细描述**：提供越详细的产品信息，效果越好
2. **明确目标**：清楚说明目标用户和使用场景  
3. **多次尝试**：尝试不同的风格和参数组合
4. **参考模板**：使用已有模板作为参考

### ✅ 高效工作流程
```
产品分析 → 生成文案 → 创作对话 → 保存模板 → 复用优化
```

### ✅ 数据管理建议
- **定期备份**：导出重要的模板和历史记录
- **分类整理**：给模板添加清晰的标签和分类
- **清理维护**：定期清理不需要的历史记录

## ⚠️ 注意事项

### 数据安全
- 所有数据存储在本地浏览器中
- 请勿清除浏览器数据，否则会丢失所有内容
- 重要数据请及时导出备份

### API使用
- 不同AI服务商的收费标准不同
- 建议根据使用频率选择合适的服务商
- 定期检查API使用量和余额

### 内容合规
- 生成的内容仅供参考，使用前请人工审核
- 确保内容符合相关法律法规
- 避免包含敏感或不当信息

## 🆘 遇到问题？

### 常见问题解决
1. **API连接失败**
   - 检查网络连接
   - 确认API密钥正确
   - 尝试更换服务商

2. **生成内容质量差**
   - 提供更详细的输入信息
   - 尝试不同的风格参数
   - 参考优质模板

3. **页面加载慢**
   - 清理浏览器缓存
   - 关闭不必要的标签页
   - 检查网络状况

### 获取帮助
- 查看完整版《产品使用指南》
- 在系统设置中查看帮助文档
- 通过反馈功能提交问题
